# About

Hi, I'm Marco.

This site is a collection of notes, experiments, and reflections while studying modern JavaScript and software engineering. The goal is simple: document what I learn and clarify complex topics by writing about them.

Most of the content here focuses on JavaScript fundamentals, runtime behavior, and practical engineering concepts that are useful for building real-world applications.

This is not meant to be a tutorial site. Instead, it's a place where I break down concepts, explore edge cases, and share insights gathered through study and experience.

## Why this site exists

Writing things down is one of the best ways to truly understand them. By documenting what I learn, I can:

- reinforce important concepts
- build a personal knowledge base
- share useful insights with other developers

## Topics you may find here

- JavaScript internals
- language fundamentals
- asynchronous programming
- debugging techniques
- software engineering principles

## About the project

This site is built with Next.js and uses markdown files as the content source. The goal is to keep the structure simple, transparent, and easy to maintain.
