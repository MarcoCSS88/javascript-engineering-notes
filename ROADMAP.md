# Roadmap

This document outlines the planned evolution of the project.

The goal is to progressively improve the blog from a simple markdown-based system to a more structured and feature-rich platform.

---

## Phase 1 — Core Functionality (Completed)

- [x] Setup Next.js project
- [x] Implement homepage layout
- [x] Add dynamic routing with `[slug]`
- [x] Load content from markdown files
- [x] Render markdown content
- [x] Add first blog post (closures)
- [x] Implement basic typography styling

---

## Phase 2 — Content & UX Improvements

- [ ] Sort posts by date (newest first)
- [ ] Improve homepage layout and spacing
- [ ] Improve single post page layout
- [ ] Add better typography consistency across pages
- [ ] Add navigation between posts (previous / next)

---

## Phase 3 — Content Structure

- [ ] Add frontmatter validation
- [ ] Introduce tags system
- [ ] Create tag-based filtering page
- [ ] Add reading time calculation
- [ ] Support for featured posts

---

## Phase 4 — Developer Experience

- [ ] Extract content loader into reusable module (`lib/content.ts`)
- [ ] Improve type safety for post data
- [ ] Add error handling for missing content (404 page)
- [ ] Improve project structure and separation of concerns

---

## Phase 5 — Advanced Features

- [ ] Add syntax highlighting for code blocks
- [ ] Add search functionality
- [ ] Add RSS feed
- [ ] Add sitemap for SEO
- [ ] Add dark mode improvements

---

## Phase 6 — Content Expansion

- [ ] Write posts on:
  - Event Loop
  - JavaScript Scope
  - Promises vs Async/Await
  - Prototypes and Object Model

---

## Long-term Vision

- Turn the blog into a high-quality engineering knowledge base
- Share content publicly to build visibility and credibility
- Use the project as a portfolio to reach senior-level opportunities
